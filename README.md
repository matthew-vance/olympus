# Olympus

All the code and documentation for my homelab.

## Controller Host Requirements

The controller is the machine that will be used to manage the homelab. I currently use a Macbook Pro, but any Linux or Mac machine should work.

### Software

- [Terraform](https://developer.hashicorp.com/terraform/downloads)
- [Ansible](https://docs.ansible.com/ansible/latest/installation_guide/index.html)

## Hypervisors

All hypervisors are running [Proxmox](https://www.proxmox.com/en/proxmox-virtual-environment/overview) VE. There are two hosts currenty, Zeus and Hera, and they are provisioned manually.
